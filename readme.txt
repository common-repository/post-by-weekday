Posting by day of the week is an easy way to manage posts

== Description ==

Post by day of the week is an easy way to manage posts.
It allows you to create a post for each day of the week.

== How to use ==
- Create a new post
- Set the post date to a specific day of the week (if not set it will be all day)
- Publish the post

== Installation ==
1. Download the plugin from https://wordpress.org/plugins/post-by-day-of-the-week/
2. Upload the plugin to the plugins directory
3. Activate the plugin
